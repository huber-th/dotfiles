# dotfiles

This is a collection of dotfiles I use to configure and personalize my system.

In order for the bootstrap to work the repo needs to be clone at `~/Developer/github/huber-th/dotfiles/`

## Credits

### nvim

My nvim configuration started as a copy of [kickstart-modular](https://github.com/dam9000/kickstart-modular.nvim)
by [dam9000](https://github.com/dam9000) which is a fork of [kickstart.nvim](https://github.com/nvim-lua/kickstart.nvim) by [nvim-lua](https://github.com/nvim-lua)

By now my configuration has been re-done from scratch. For someone new to nvim I highly recommend to check out the above repositories as they have helped me a great deal in getting started.
